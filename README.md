# snu-menu
매일매일 정해진 시간에 서울대학교 메뉴를 파싱하여 정해진 시간에 봇으로 메뉴를 공지할 수 있는 간단한 봇

# 적용방법
1. 구글 스프레드 시트를 하나 만드세요.
2. 확장 프로그램 -> Apps Script를 클릭하세요.
3. 코드를 입력하고 저장하세요.
4. 이 때, 코드 내에 링크하고 싶은 봇의 웹후크 주소로 수정하세요.
5. update를 실행하세요.
6. build_slack_message를 실행해 Log에 잘 찍히는지 확인하세요.
7. 좌측 트리거 메뉴에서 다음과 같이 트리거를 추가하세요.
    |실행할 함수 선택|실행할 배포 선택|이벤트 소스 선택|트리거 기반 시간 유형 선택|시간 선택|
    |:---:|:---:|:---:|:---:|:---:|
    |update|Head|시간 기반|일 단위 타이머|오전 8시~오전 9시 사이|
    |update|Head|시간 기반|일 단위 타이머|오전 9시~오전 10시 사이|
    |sendmsg|Head|시간 기반|일 단위 타이머|오전 10시~오전 11시 사이|
    |sendmsg|Head|시간 기반|일 단위 타이머|오후 4시~오후 5시 사이|
8. 좌측 설정 메뉴에서 시간대를 한국 표준시 - 서울로 지정하세요.
